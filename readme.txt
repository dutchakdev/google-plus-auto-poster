**This plugin automatically publishes posts from your blog to your Social Network accounts**

announcements of your posts to most popular Social Network site Google Plus. With the Use of GPAP you can save your valuable

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wpsp-googleplus` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Alternatively, you can install and activate the plugin through the WordPress plugin manager.

== Frequently Asked Questions ==

= What is the difference between "Free" and "Pro" versions? =

Free Plugin is limited access means only automatically post on google profile page and when Pro plugin is not only automatically
post on google profile page with google business page and google communites page

= Why google plus auto poster are not fully free? =

Google+ and Pinterest don't have API or have a read-only API that doesn't allow to make posts. We had to create our own API 
libraries for google+ networks and those libraries are available in google plus auto poster
